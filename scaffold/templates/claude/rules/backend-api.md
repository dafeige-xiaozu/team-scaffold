# 后端开发规范

- 改完确认 pytest 无新增失败
- 改接口必须先更新 contracts/CONTRACTS.md
- 新会话先读 backend/STATUS.md
- 前端需求拒绝：「这是前端任务，请发给{{role_frontend}}」

## 适用角色
本规则适用于{{role_backend}}（后端工程师）。
{{role_devops}}（联调负责人）为联调目的修改 backend/ 文件时，遵循{{role_devops}} agent 定义中的跨目录修改权限，不受本规则的"拒绝"条款约束。
