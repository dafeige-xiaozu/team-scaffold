# /assign-task

辅助{{role_architect}}生成标准格式的任务提示词，发给指定角色执行。

## 提示词模板

生成的提示词必须包含以下结构：

```
任务：{{一句话描述}}

可与 XXX 并行 / 需串行等 XXX 完成后启动

需要读的文件

- {{文件路径 1}}
- {{文件路径 2}}

背景

{{任务背景说明}}

改动清单

{{具体步骤}}

验证标准（必须覆盖 L4 深度）

功能验证：
- {{功能正确}}

持久化验证：
- F5 刷新后 {{状态保持}}

逆向验证：
- 取消/删除后 {{状态回退}}

注意：全程自主完成，不要中途停下来问我。遇到不确定的选最保守方案，完成后在汇报里说明你的决策理由。
```
